This readme should explain how to run the code for assignment 1 CS7641 (Machine learning)

The required packages are:
	-Numpy
	-Pandas
	- Matplotlib
	- Seaborn 
	- tqdm
	- scikit-learn

The code can be accessed via this link:  https://github.gatech.edu/ajar3/ML_GT

There are two main Jupyter notebook (one for each dataset). They can be found under  assignment 1/code folder
	-Heart attach prediction.ipynb 
	-letter.ipynb

The notebooks are divided into sections (exploring the data, preprocessing, ANN, SVM, DT, Boosting, KNN, and conclusion).
